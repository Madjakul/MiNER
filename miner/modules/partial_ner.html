class miner.modules.partial_ner.PartialNER(lm_path: str, num_labels: int, device: Literal['cpu', 'cuda'], dropout: float, q: float | None = None, padding_idx: int | None = None)[source]

Partial Named Entity Recognizer (NER) Model.

This class defines a Partial NER model for named entity recognition tasks. It extends the PyTorch nn.Module class and integrates with the Hugging Face transformers library for handling pre-trained language models.

Parameters:

  • lm_path (str) – The path or identifier of a pre-trained language model checkpoint.

  • num_labels (int) – The number of unique labels or tags for NER.

  • device (str, {"cpu", "cuda"}) – The device on which the model will be instantiated (“cpu” or “cuda”).

  • dropout (float) – The dropout probability to apply to the model’s hidden states.

  • q (float, optional) – The q-value for the partial CRF layer. If None, no partial CRF is used.

  • padding_idx (Optional[int], optional) – The padding index for the input sequences. If None, the default index is used.
